# General Documents

Bu klasor sprintlerden bagimsiz proje kararlarini ve genel referanslari takip etmek icin olusturulmustur.

## Proje Ozeti

- Urun adi: SkinShelf
- Alan: Yapay zeka destekli cilt bakim urun dolabi ve rutin asistanı
- Platform: React Native / Expo / TypeScript mobil uygulama
- Hedef backend: Java / Spring
- Hedef AI entegrasyonu: Gemini 2.5 Flash
- Acik urun verisi: Open Beauty Facts

## Sprint Yol Haritasi

| Sprint | Odak |
| --- | --- |
| Sprint 1 | Mobil prototip, temel ekranlar, Scrum dokumantasyonu, teknik mimari |
| Sprint 2 | Backend, database, barkod verisi, AI analiz entegrasyonu |
| Sprint 3 | Gelismis rutin onerileri, bildirimler, test, final demo |
