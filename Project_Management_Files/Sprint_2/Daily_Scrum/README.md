# Sprint 2 Daily Scrum

Sprint 2 boyunca takim iletisimi WhatsApp ve sesli gorusmeler uzerinden yurudu. Ornek repolardaki gibi daily scrum kanitlari tek bir link ve ozet dosyasi uzerinden referanslandi.

## Kanit Linki

Daily scrum konusmalari ve toplantilar: [Sprint 2 Daily Scrum Chats](https://imgur.com/a/7xPVw1X)

## Ele Alinan Basliklar

- Spring Boot backend ayaga kaldirma ve port/CORS problemleri
- Supabase baglantisi, SSL ayarlari ve migration duzeni
- Kayit/giris akisi ve onboarding verisinin kullanici ID'siyle eslestirilmesi
- Barkod okutma ve Open Beauty Facts'ten urun bilgisi cekme
- Gemini kota sinirlari ve fallback cevabi
- Shelly'nin son mesajlari hatirlamasi
- Dolapta aktif/pasif kullanilan urunlerin Rutinim ekranina yansimasi
- Cilt takibi, fotograf notu ve haftalik ozet akisi

## Kararlar

| Konu | Karar |
| --- | --- |
| Backend framework | Java/Spring Boot ile devam edildi. |
| Database | Supabase PostgreSQL kalici veri katmani olarak kullanildi. |
