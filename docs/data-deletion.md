# SkinShelf Data Deletion

Last updated: 2026-07-12

Users can request deletion directly in the app:

1. Open Profile.
2. Tap "Hesabı ve Verileri Sil".
3. Confirm the destructive action.

This deletes the account, skin profile, shelf products, assistant chat history, and skin tracking logs from the backend database.

For store listing or external support workflows, publish this document as the public data deletion URL.
